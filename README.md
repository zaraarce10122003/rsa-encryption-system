# RSA Encryption System

This project implements the RSA public-key encryption algorithm in C, including key generation, encryption, and decryption.

## Features
- Recursive GCD calculation
- Modular inverse using the extended Euclidean algorithm
- RSA key generation
- Modular exponentiation for encryption and decryption

## How to Run
```bash
gcc rsa.c -o rsa
./rsa
